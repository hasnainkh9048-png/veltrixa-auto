SETUP INSTRUCTIONS:
-------------------
1. Create a folder called "veltrixa-auto-website" on your computer
2. Create all subfolders (css, js, images)
3. Copy all HTML files to main folder
4. Copy style.css to css folder
5. Copy firebase-config.js to js folder
6. Rename your logo to "veltrixa-logo.png" and put it in images folder
7. Update firebase-config.js with your Firebase details
8. Open index.html in your browser to test

FIREBASE SETUP:
---------------
1. Go to https://console.firebase.google.com/
2. Create new project "Veltrixa-Auto"
3. Enable Firestore Database
4. Enable Authentication (Email/Password)
5. Create admin user in Authentication
6. Get your Firebase config and update firebase-config.js
